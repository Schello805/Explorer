# Beiträge

Danke für dein Interesse an Platzguide.

1. Issue mit Problem und gewünschtem Verhalten anlegen.
2. Kleine, fokussierte Änderung erstellen.
3. `npm run verify` erfolgreich ausführen.
4. Pull Request mit Test- und Datenschutzfolgen beschreiben.

Beiträge werden unter derselben nicht-kommerziellen Lizenz veröffentlicht.
Bitte keine echten Gästedaten, Zugangsdaten oder ungeklärten Bildrechte
einreichen.
